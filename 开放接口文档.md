# 开放接口，token


## 获取token信息


**接口地址**:`/open/token`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:<p>登录后使用code来换取token信息，获取的是当前应用的数据</p>



**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|appId|应用id，必填|query|false|string||
|authorityCode|接口提供者的权限标识|query|false|string||
|clientIp|接口调用者的IP|query|false|string||
|myAppId|接口提供者的应用ID|query|false|string||
|restCode|rest接口标识|query|false|string||
|sign|签名，必填|query|false|string||
|signData|签名的数据，json格式，必须有timeStamp，randomStr两个字段，时间戳和随机支付串|query|false|string||
|signType|签名类型，必填md5：md5签名，使用BASE64编码数据。aes：aes签名，使用BASE64编码数据。aes_md5：aes和md5结合签名，数据使用aes加密，签名使用md5，适用于安全级别比较高的调用。|query|false|string||
|userId|用户id|query|false|string||
|accessToken|accessToken|query|false|string||
|code|code|query|false|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|TokenDto|
|401|Unauthorized||
|403|Forbidden||
|404|Not Found||


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|accessToken|访问令牌|string||
|account|用户账号|string||
|appId|登录的是那个应用|string||
|authorityCodes|应用的权限|array||
|data|其他数据|object||
|groups|用户组集合|array|GroupModel|
|refreshToken|表示早前收到的更新令牌|string||
|updateTime|更新时间|string(date-time)|string(date-time)|
|userId|用户Id|string||
|validTime|有效期|integer(int64)|integer(int64)|


**响应示例**:
```javascript
{
	"accessToken": "",
	"account": "",
	"appId": "",
	"authorityCodes": [],
	"data": {},
	"groups": [
		{}
	],
	"refreshToken": "",
	"updateTime": "",
	"userId": "",
	"validTime": 0
}
```
